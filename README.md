# Todo App

Приложение, которое отображает список Ваших дел. Есть возможность создавть список, удалять из него, отмечать дела как "важные" и "завершенные", а также искать нужные дела с помощью поиска.

Todos - было создано с целью изученеие библиотеки React JS

Стек: React, Redux, Bootstrap, localStorage

## Клонирование и установка
Вам понадобится <code>[node](https://nodejs.org/en/)</code> и <code>[npm](https://www.npmjs.com/)</code>. Вы можете использовать <code>[yarn](https://yarnpkg.com/en/)</code> вместо <code>npm</code>.
Проделайте следующее:
```
> git clone https://github.com/ponev/todos.git
> cd todos
> npm install или yarn install
> npm start или yarn start
```

### Не знакомы с Git?
Вы можете просто скачать .zip файл. Извлеките содержимое, затем перейдите в каталог проекта, откройте свой терминал и используйте:
```
> npm install
> npm start
```
